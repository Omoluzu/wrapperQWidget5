Параметры который использует виджет **QPushButton**

&nbsp;
###### size:
Устанавливает ширину и высоту кнопки
Может принимать в себя как `list[x, y]` так и просто `xy`
где `x` - ширина кнопки
где `y` - высота кнопки
где `xy` - ширина и высота кнопки

Пример использования.
```python
"size": [x, y]
"size": [xy]
"size": xy
```
